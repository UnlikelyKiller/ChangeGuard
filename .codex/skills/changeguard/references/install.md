# ChangeGuard Installation

If ChangeGuard is not installed or not on the system `PATH`, follow these instructions.

## Linux / macOS (Bash)

```bash
curl -fsSL https://raw.githubusercontent.com/UnlikelyKiller/ChangeGuard/main/install/install.sh | sh
```

## Windows (PowerShell)

```powershell
iwr https://raw.githubusercontent.com/UnlikelyKiller/ChangeGuard/main/install/install.ps1 -UseB | iex
```

After installation, open a new terminal window or refresh your environment variables to ensure `changeguard` is available.
